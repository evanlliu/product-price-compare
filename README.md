# 商品比价器 Price Compare v1.0.59

本版本以 **v1.0.57** 为基线重新优化，放弃 v1.0.58 的“右侧预留安全区域”方案，改为让移动端右侧悬浮按钮支持自由拖动。

## v1.0.59 更新内容

1. 回退到 v1.0.57 的页面布局基线。
   - 保留 v1.0.57 的 PC 端“历史对比”列优化。
   - 不继续使用 v1.0.58 中移动端列表右侧强制留白的方案。

2. 移动端右侧悬浮按钮支持拖动。
   - 可以直接按住右侧悬浮按钮组上下移动。
   - 如果挡住文字内容，可以拖到上方或其他空白区域。
   - 拖动时不会误触发新增、同步、语言切换按钮。

3. 悬浮按钮位置按设备本地记忆。
   - 位置保存到当前浏览器 / 当前设备的 `localStorage`。
   - 不写入 `data.json`，避免不同手机、电脑之间互相覆盖位置。
   - 刷新页面后会保持上次拖动的位置。

4. 自动限制拖动范围。
   - 不允许拖出屏幕外。
   - 横竖屏切换或窗口尺寸变化后，会自动修正到可见区域内。

5. 版本号同步更新。
   - `index.html` 版本号更新为 `v1.0.59`
   - 示例 `data.json` 版本号更新为 `v1.0.59`
