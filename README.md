# 商品比价器 Price Compare v1.0.47

本版本基于 **v1.0.46** 继续优化，重点调整同步状态显示，让顶部状态更简洁。

## v1.0.47 更新内容

1. 同步状态文案精简。
   - 删除同步信息中的“云端 data.json / GitHub data.json”等字样。
   - 中文示例：`已加载：18:33:20`、`已保存：18:33:20`。
   - 英文示例：`Loaded: 18:33:20`、`Saved: 18:33:20`。

2. 同步时间格式调整。
   - 原来显示完整日期时间。
   - 现在只显示 `HH:mm:ss`。
   - 示例：`18:33:20`。

3. 移动端顶部同步信息同步优化。
   - 同步信息继续显示在版本号后面。
   - 文案更短，减少移动端顶部占用。

4. 保留 v1.0.46 功能。
   - 移动端新增按钮为右侧悬浮按钮。
   - 移动端记录操作支持左滑显示。
   - 左滑操作展开时隐藏右侧悬浮按钮。
   - 移动端“更多”按钮为图标显示。
   - 移动端主页面商品分组使用原生下拉选择。

## GitHub 更新文件

需要上传 / 覆盖：

- `index.html`
- `README.md`

通常保留你线上已有的：

- `data.json`

不要用压缩包里的示例 `data.json` 覆盖线上真实数据，除非你确认要重置数据。
