# 商品比价器 Price Compare v1.0.46

本版本基于 **v1.0.44** 继续迭代，重点优化移动端价格记录展示密度，让手机上一屏可以看到更多历史价格数据。

## v1.0.46 更新内容

1. 移动端价格记录改为紧凑查看卡片。
   - 不再默认显示大量输入框。
   - 每条记录优先显示商品名称、商家 / 渠道、商品分组、日期、状态。
   - 一屏可以看到更多历史记录，方便快速比较价格。

2. 移动端重点数据前置。
   - 每条卡片直接显示：
     - 每单位价格
     - 实际金额
     - 总规格
     - 历史对比
   - 可以更快判断这次价格是贵了还是便宜了。

3. 移动端编辑方式调整。
   - 点击卡片空白区域可以打开编辑弹窗。
   - 点击“编辑”按钮也可以打开编辑弹窗。
   - 移动端不再在卡片内直接编辑，减少误触和页面高度占用。

4. 移动端新增快速筛选标签。
   - 全部
   - 本次待买
   - 已购买
   - 历史最低

5. PC 端保持原有表格逻辑。
   - PC 表格行内展示和编辑逻辑不变。
   - 仍保留鼠标悬停显示操作按钮的交互。

6. 保留 v1.0.44 修复。
   - 配置弹窗 / 编辑弹窗层级正常。
   - 更多菜单不再被当前商品分组区域遮挡。
   - 打开弹窗不会让主页面跳到顶部。
