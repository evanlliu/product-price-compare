# 商品比价器 Price Compare v1.0.57

本版本基于 **v1.0.56** 继续迭代，重点优化 PC 端表格中“历史对比”列的展示方式，解决信息被截断、可读性差的问题。

## v1.0.57 更新内容

1. 重新设计 PC 端“历史对比”列。
   - 不再只用一整条长胶囊文字显示。
   - 改为更清晰的分层结构：
     - 第一层：状态标签，如 `历史最低` / `上次购买`
     - 第二层：核心比较信息，如 `比历史最低便宜 0.084/片`
     - 第三层：总价差额，如 `总价便宜 17.57`

2. 解决历史对比文字显示不全的问题。
   - 历史对比列宽度加大。
   - 该列支持自然换行，不再因为列宽过窄而把文字裁掉。

3. 保留不同类型信息的视觉层级。
   - 简短状态继续用标签显示。
   - 详细对比信息改为文字说明。
   - 总价差额保留为单独的小高亮标签，更适合作为购买判断重点。

4. 仅优化 PC 表格中的“历史对比”列。
   - 移动端卡片展示逻辑保持不变。

5. 版本号同步更新。
   - `index.html` 版本号更新为 `v1.0.57`
   - 示例 `data.json` 版本号更新为 `v1.0.57`
