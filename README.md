# 商品比价器 Price Compare v1.0.34

本版本基于 **v1.0.33** 继续迭代，重点调整价格记录中“总规格”和“每单位价格”的显示格式。

## v1.0.34 更新内容

1. 总规格列显示格式调整
   - PC 端表格中的“总规格”改为：`总规格/单位`。
   - 示例：`360/g`、`1.5/kg`、`500/ml`。
   - 鼠标悬停提示也同步显示完整格式。

2. 每单位价格列显示格式调整
   - PC 端表格中的“每单位价格”改为：`每单位价格/单位`。
   - 示例：`0.182/g`、`12.500/kg`、`0.035/ml`。
   - 每单位价格继续保留 3 位小数。

3. 移动端同步调整
   - 移动端卡片里的“总规格”也同步改为 `总规格/单位`。
   - “每单位价格”原本已经按 `价格 / 单位` 展示，本次统一为更紧凑的 `价格/单位`。

4. 保留 v1.0.33 功能
   - 单位配置弹窗一页显示。
   - 单位配置支持“关闭 / 保存”后再写入。
   - 右侧云端悬浮按钮用于重新加载云端 `data.json`。
   - 线上部署环境优先读取同目录 `data.json`。

## GitHub 更新文件

需要上传 / 覆盖：

- `index.html`
