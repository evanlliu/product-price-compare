# 商品比价器 Price Compare v1.0.60

本版本基于 **v1.0.59** 继续迭代，按设备区分悬浮按钮拖动逻辑，并优化顶部同步状态的时间显示。

## v1.0.60 更新内容

1. PC 端悬浮按钮恢复固定位置。
   - PC 端不再支持拖动悬浮按钮。
   - 保持原来的右下角固定显示，避免鼠标操作时误拖动。

2. 移动端继续支持拖动悬浮按钮。
   - 手机端可以按住右侧悬浮按钮组拖动。
   - 如果挡住内容，可以移动到上方或其他空白位置。
   - 位置只保存在当前设备本地，不写入 `data.json`，不同设备互不影响。

3. PC / 移动端自动切换悬浮按钮行为。
   - 当页面宽度进入移动端布局时，启用拖动位置。
   - 回到 PC 端布局时，自动清除内联定位，恢复固定位置。

4. 优化同步状态显示文案。
   - `已加载云端 data.json：2026/5/14 17:48:31` 改为 `已加载：17:48:31`
   - `已保存到 GitHub data.json：17:48:31` 改为 `已保存：17:48:31`

5. 同步时间格式统一为 `HH:mm:ss`。
   - 例如：`17:48:31`
   - 不再显示日期。

6. 版本号同步更新。
   - `index.html` 版本号更新为 `v1.0.60`
   - 示例 `data.json` 版本号更新为 `v1.0.60`
