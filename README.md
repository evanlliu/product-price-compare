# 商品比价器 Price Compare v1.0.44

本版本基于 **v1.0.37** 继续迭代，重点优化商品分组选择、商品分组配置排序，以及价格记录编辑方式。

## v1.0.44 更新内容

1. 修复点击“更多”、打开配置弹窗、打开编辑弹窗时主页面跳到顶部的问题。
2. 打开弹窗时会锁定并保留当前页面滚动位置。
3. 关闭弹窗后自动回到打开前的位置，不会再出现背景页面回到顶部的体验问题。
4. `Esc` 关闭配置弹窗和编辑弹窗时也会正确恢复滚动位置。

1. 商品分组配置支持拖动排序。
   - 商品分组配置页面每一行左侧新增拖动按钮。
   - 拖动分组行即可调整商品分组显示顺序。
   - 排序结果会保存到 `data.json`。

2. 主页面“当前商品”改名为“当前商品分组”。
   - 中文显示：`当前商品分组`。
   - 英文显示：`Current Product Group`。

3. 当前商品分组选择支持模糊查询。
   - 主页面不再只是普通下拉选择。
   - 输入商品分组关键字即可过滤匹配项。
   - 点击匹配项或按 Enter 可切换当前商品分组。

4. 编辑按钮逻辑优化。
   - 不管记录是“本次待买”还是“已购买”，PC 端鼠标放到该行上都会显示编辑按钮。
   - 点击编辑后进入编辑弹窗。
   - 编辑弹窗中可以修改该记录所属的商品分组。
   - 如果把记录移动到其他商品分组，保存后记录会移动到目标分组。

5. 删除码数列。
   - 表格不再显示“码数”列。
   - 移动端卡片也不再显示“码数”。
   - 编辑弹窗不再保存码数字段。
   - 后续保存到 `data.json` 时会自动清理旧数据中的 `sizeCode / codeSize / code` 字段。

6. 保留 v1.0.37 功能。
   - 页面刷新时优先加载 `data.json` / 云端数据，避免旧缓存先闪一下。
   - 历史对比显示金额差，不显示百分比。
   - 总规格显示如 `360克`。
   - 每单位价格显示如 `0.182/克`。
   - 右侧云端悬浮按钮用于重新加载云端 `data.json`。
