# 商品比价器 Price Compare v1.0.31


## v1.0.31 更新

1. 修复单位配置保存问题
   - 单位配置页面现在使用临时草稿。
   - 修改、新增、删除单位后，只有点击底部“保存”才会写入当前数据。
   - 点击“关闭”、右上角关闭或弹窗外关闭会放弃本次未保存改动。

2. 修复保存到 data.json 的逻辑
   - 点击单位配置“保存”后会更新 `state.units`。
   - 同步更新 `configs.units`。
   - 如果配置了 Cloudflare Worker，会立即保存到 GitHub `data.json`，不是只保存到本地。

3. 保留现有功能
   - 右下角 `btnFloatingSync` 仍然用于重新拉取云端 `data.json` 到当前设备。
   - 其他配置仍然通过 `saveState()` 自动保存并同步。

### GitHub 更新文件

- `index.html`
- `README.md`

### Cloudflare Worker

不需要更新。

## 本版本更新

1. PC 端表格内容显示优化
   - 表格改为按内容自适应宽度，不再强行压缩单元格。
   - 金额、返现、购买数量、单件规格、码数等输入框加宽并支持输入时自动扩展。
   - 日期、单位、每单位价格等列重新调整，尽量完整显示内容。

2. PC 端操作按钮显示修复
   - 操作列改为右侧固定显示区域。
   - 鼠标放到行上时，“买了 / 编辑 / 复制 / 删除”按钮会完整显示。
   - 操作按钮不再被右侧边缘裁切。

3. 表格横向体验优化
   - 如果内容确实超过屏幕宽度，表格允许横向滚动。
   - 操作列保持在右侧，避免滚动后找不到操作按钮。

4. 继续保留 v1.0.25 功能
   - 商品名称配置页面布局优化。
   - 已购买记录通过弹窗编辑，避免误改。
   - 商品名称维护，并和商品分组绑定。
   - 单位配置维护。
   - 每单位价格保留 3 位小数。
   - 本地 debug 支持通过 Cloudflare Worker 同步云端 data.json。
   - PC + 移动端适配。
