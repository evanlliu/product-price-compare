# 商品比价器 Price Compare v1.0.29


## v1.0.29

- PC 端单位配置页面改为紧凑表格布局。
- 单位配置的编码、中文、英文、基础单位、换算系数在 PC 端同一行维护。
- 移动端继续保持适合触控的卡片式编辑。

## 本版本更新

1. PC 端表格内容显示优化
   - 表格改为按内容自适应宽度，不再强行压缩单元格。
   - 金额、返现、购买数量、单件规格、码数等输入框加宽并支持输入时自动扩展。
   - 日期、单位、每单位价格等列重新调整，尽量完整显示内容。

2. PC 端操作按钮显示修复
   - 操作列改为右侧固定显示区域。
   - 鼠标放到行上时，“买了 / 编辑 / 复制 / 删除”按钮会完整显示。
   - 操作按钮不再被右侧边缘裁切。

3. 表格横向体验优化
   - 如果内容确实超过屏幕宽度，表格允许横向滚动。
   - 操作列保持在右侧，避免滚动后找不到操作按钮。

4. 继续保留 v1.0.25 功能
   - 商品名称配置页面布局优化。
   - 已购买记录通过弹窗编辑，避免误改。
   - 商品名称维护，并和商品分组绑定。
   - 单位配置维护。
   - 每单位价格保留 3 位小数。
   - 本地 debug 支持通过 Cloudflare Worker 同步云端 data.json。
   - PC + 移动端适配。

## v1.0.29

- PC 端删除可见的“操作”列标题与占位宽度。
- 行操作按钮改为悬浮显示，不再占用表格固定列宽。
- PC 端表格列宽重新压缩，优先一屏显示完整内容，减少横向滚动条。


## v1.0.29 更新

1. 单位配置弹窗在 PC 端改为一页紧凑显示，默认单位配置不再出现弹窗内部滚动条。
2. 右下角悬浮云端按钮 `btnFloatingSync` 改为重新拉取云端 / data.json 数据，并重新加载到当前设备。
3. 悬浮云端按钮不再执行保存动作，避免误把当前本地数据覆盖到云端。
