# 商品比价器 Price Compare v1.0.37

本版本基于 **v1.0.36** 继续迭代，重点优化刷新页面时先显示本机缓存旧数据、随后又跳到云端最新数据的问题。

## v1.0.37 更新内容

1. 页面启动流程优化。
   - 页面打开 / 刷新时先显示加载界面。
   - 优先读取 `data.json` / 云端数据。
   - 云端数据加载完成后再渲染主页面。

2. 解决刷新时“旧缓存闪一下”的体验问题。
   - 不再先渲染本机 `localStorage` 旧数据。
   - 避免先看到旧记录，几秒后又突然变成最新记录。

3. 本地 debug 逻辑保留。
   - 本地 `file://`、`localhost`、`127.0.0.1` 环境仍然可以使用本机保存过的 Cloudflare 配置去加载云端数据。
   - 如果本地没有配置过同步信息，会在加载结束后进入本地模式。

4. 保留 v1.0.36 功能。
   - 历史对比显示金额差，不显示百分比。
   - 总规格显示为 `360克`。
   - 每单位价格显示为 `0.182/克`。
   - 右侧云端悬浮按钮用于重新加载云端 `data.json`。
