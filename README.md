# 商品比价器 Price Compare v1.0.22

## 本版重点

v1.0.22 是在 v1.0.18 基础上重新合并并修复后的版本，保留 v1.0.18 的配置功能，同时继续加入后续需求。

## 更新内容

1. 修复 v1.0.18 相关功能缺失问题
   - 商品名称维护已重新实现。
   - 商品名称与商品分组绑定。
   - 价格记录表格中可下拉选择商品名称。
   - 单位配置维护已重新实现。

2. 商品名称配置
   - 更多 → 商品名称配置。
   - 当前商品分组下可以维护多个商品名称。
   - 每条价格记录选择对应商品名称。
   - 历史对比按“商品名称 + 单位类型”分组，避免不同商品互相干扰。

3. 单位配置
   - 更多 → 单位配置。
   - 可维护单位编码、中文名称、英文名称、基础单位、换算系数。
   - 基础单位相同才会互相比较，例如 kg 与 g 可以比较。

4. 已购买记录锁定
   - 状态为“已购买”的记录不可编辑。
   - 需要修改时，先把状态释放回“本次待买”。

5. 每单位价格保留三位小数
   - PC 表格、移动端卡片、顶部推荐条统一保留 3 位小数。

6. 本地 debug 云端同步
   - 支持 GitHub Pages、localhost、127.0.0.1、file:// 场景。
   - 本机浏览器会保存 Cloudflare Worker 地址和密码，用于本地调试时读取云端 data.json。
