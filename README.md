# 商品比价器 Price Compare v1.0.66

本版本基于 **v1.0.65** 继续迭代，重点优化顶部“当前最便宜”推荐条中总价标签的状态语义。

## v1.0.66 更新内容

1. 总价标签颜色跟随当前推荐状态。
   - 绿色：历史低价 / 可以买 / 接近上次。
   - 黄色：略贵，可以观望。
   - 红色：偏贵，建议等等。
   - 不再固定使用蓝色表示“上次总价”、青绿色表示“历史最低总价”。

2. 总价差额保持更高视觉层级。
   - 单位差价使用浅色标签。
   - 总价差额使用深色实心标签。
   - 金额数字继续放大加粗，方便快速判断本次总价影响。

3. “接近”状态单独处理。
   - 单位价格接近时显示：`接近上次`、`接近历史最低`。
   - 总价接近时显示：`上次总价接近`、`历史最低总价接近` 或合并后的 `总价接近`。
   - 避免接近状态下还显示 `贵 0.01` 这类容易造成误解的文案。

4. 自动合并重复比较信息。
   - 当“上次购买”和“历史最低”是同一个价格参考时，不再重复显示两套完全一样的比较。
   - 例如会合并成：`比上次/历史最低贵 0.096/片` + `总价贵 20.01`。

5. 版本号同步更新。
   - `index.html` 版本号更新为 `v1.0.66`
   - `app.js` 版本号更新为 `v1.0.66`
   - 示例 `data.json` 版本号更新为 `v1.0.66`
