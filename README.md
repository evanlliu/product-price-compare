# 商品比价器 Price Compare v1.0.36

本版本基于 **v1.0.35** 继续迭代，主要优化历史价格提示方式。

## v1.0.36 更新内容

1. 顶部购买建议不再显示百分比。
   - 原来：`比上次贵 6.73%`
   - 现在：`比上次贵 8.45/罐`

2. 表格中的“历史对比”标签不再显示百分比。
   - 原来：`比上次贵 6.73%`
   - 现在：`比上次贵 8.45/罐`

3. 历史平均价、历史最低价对比也改成金额差。
   - 示例：`比历史平均贵 6.25/罐`
   - 示例：`比历史最低便宜 0.012/克`

4. 每单位价格继续保留 3 位小数。

5. 保留 v1.0.35 功能。
   - 总规格显示为 `360克`，不使用 `/`。
   - 每单位价格显示为 `0.182/克`。
   - 单位配置支持关闭 / 保存后再写入。
   - 右侧云端悬浮按钮用于重新加载云端 `data.json`。
   - 线上部署环境优先读取同目录 `data.json`。

## GitHub 更新文件

需要上传 / 覆盖：

- `index.html`
- `README.md`

通常保留你线上已有的：

- `data.json`

不要用压缩包里的示例 `data.json` 覆盖线上真实数据，除非你确认要重置数据。
